# 1. 产品概述

## 1.1 产品名称

小型企业费用报销管理系统（Expense Reimbursement Management System）

## 1.2 产品背景

传统纸质报销流程存在审批效率低、数据统计困难、流程不透明等问题。

本系统通过信息化手段，实现企业报销申请、审批、付款和归档全流程线上管理，提高财务工作效率，降低管理成本，实现预算前置控制。

## 1.3 产品目标

- 实现报销流程电子化
- 提高审批效率
- 实现费用统计与数据分析
- 支持预算控制和权限管理
- 提供完整的审计日志
- 支持后续扩展工作流

------

# 2. 用户角色

| 角色       | 职责                         |
| ---------- | ---------------------------- |
| 员工       | 提交报销申请、查看审批状态   |
| 部门主管   | 审批报销申请、查看部门统计   |
| 财务人员   | 财务审核、付款、归档         |
| 系统管理员 | 用户管理、流程配置、系统维护 |

------

# 3. 产品流程

```text
员工提交申请
      ↓
部门主管审批
      ↓
财务审核
      ↓
财务付款
      ↓
自动归档
```

异常流程：

```text
主管驳回
↓
员工修改重新提交

财务驳回
↓
员工修改重新提交
```

# 4. 功能模块

## 4.1 登录认证模块

### 功能描述

用户通过账号密码登录系统。

### 功能列表

- 登录
- 退出登录
- 修改密码
- Token认证

------

## 4.2 报销申请模块

### 功能描述

员工在线提交报销申请。

### 功能列表

- 新建报销单
- 保存草稿
- 提交申请
- 编辑申请
- 撤销申请
- 上传电子凭证

### 输入信息

- 报销类型
- 金额
- 报销事由
- 凭证附件

### 状态

- 草稿
- 待主管审批
- 待财务审核
- 待付款
- 已付款
- 驳回

------

## 4.3 主管审批模块

### 功能描述

主管审核下属提交的报销申请。

### 功能列表

- 查看待审批列表
- 查看报销详情
- 查看凭证
- 审批通过
- 驳回申请
- 填写驳回原因
- 查看审批历史

------

## 4.4 财务审核模块

### 功能描述

财务人员审核申请并完成付款。

### 功能列表

- 查看待审核列表
- 审核报销单
- 查看原始凭证
- 审核通过
- 驳回申请
- 录入付款信息
- 标记付款状态
- 自动归档

付款信息包括：

- 付款方式
- 付款时间
- 银行流水号

------

## 4.5 统计分析模块

### 功能列表

#### 个人统计

- 月度报销金额
- 年度报销金额

#### 部门统计

- 部门报销总额
- 按员工统计

#### 企业统计

- 月度统计
- 季度统计
- 按费用类型统计

#### 图表展示

- 柱状图
- 折线图
- 饼图

#### 数据导出

支持 Excel 导出。

- 下载附件


## 4.11 审批流程配置

支持：

### 普通流程

```text
主管 → 财务
```

### 多级流程

```text

 ↓
高级主管
 ↓
财务
```

# 6. 非功能需求

## 性能要求

- 页面响应时间 < 2 秒
- 支持 100 个用户同时在线

## 安全要求

- JWT身份认证
- BCrypt密码加密
- RBAC权限控制
- SQL注入防护

## 可维护性

采用前后端分离架构：

```text
Vue3
↓
Spring Boot
↓
MySQL
```

## 可扩展性

支持：

- Redis缓存
- Flowable工作流
- Docker部署
- 消息队列

# 7. 产品特色

## 特色一：财务一键拒绝

支持：

- 一键驳回
- 拒绝原因模板
- 自动通知员工
- 审批记录可追溯

价值：

提高审核效率，加强合规管理。

------

## 特色二：部门预算前置控制

主管可根据本月数据设置下月预算额度。

系统支持：

- 超额预警
- 超额限制
- 动态额度调整

价值：

实现主动预算管理，降低企业成本。

# 10. 项目目标

构建一个面向小型企业的费用报销管理平台，实现申请、审批、付款、归档全过程数字化管理，提高审批效率，强化预算控制，为企业提供可靠、安全、可扩展的费用管理解决方案。
